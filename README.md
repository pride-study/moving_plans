# LGBTQIA+ people’s moving thoughts, plans, decisions following 2024 U.S. Presidential Election 

## Overview

This repository contains all analytical code for a cross-sectional study examining LGBTQIA+ individuals' moving plans and decisions following the 2024 U.S. Presidential election. The study analyzes data from The PRIDE Study's annual questionnaires (AQ2024 and AQ2025), focusing on responses collected from participants who completed questionnaires after November 4, 2024.

## Data Sharing

De-identified quantitaive data are available upon reasonable request through The PRIDE Study's Ancillary Study application process. Researchers interested in The PRIDE Study data may submit a brief application which is reviewed by both a Research Advisory Committee (composed of scientists) and Participant Advisory Committee (composed of participants) to affirm appropriate data use. Details about the Ancillary Study process are available at [www.pridestudy.org/collaborate](https://www.pridestudy.org/collaborate) or by contacting [support@pridestudy.org](mailto:support@pridestudy.org) or 855-421-9991 (toll-free). Qualitative data (write-in responses) are not available due to ethical restrictions related to sensitive participant information. 

## Technical Requirements 

The code is written in R version 4.5.1, and we use renv to facilitate reproducibility. Primary packages required for analysis are tidyverse (version 2.0.0), here (1.0.1), psych (2.6.1), marginaleffects (0.27.0), and tableone (0.13.2).
